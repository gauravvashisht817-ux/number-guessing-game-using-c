# Number Guessing Game in C 🎮

This is a simple number guessing game written in C.

The computer generates a random number between 1 and 100 and the player must guess it within limited attempts.

## Features
- Random number generation
- Three difficulty levels
- Limited attempts
- Hints (Too High / Too Low)
- Play again option

## Difficulty Levels
Easy - 10 attempts  
Medium - 7 attempts  
Hard - 5 attempts  

## How to Run

Compile the program

gcc guessing_game.c

Run the program

./a.out
